# ChipBook

## 🚀 바로 보기

👉 **[ChipBook 열기](https://bujh99.github.io/ChipBook/)**

## 📚 구성

- 🧭 반도체 산업과 설계 흐름 개요
- 🧩 RTL / Frontend 핵심 개념
- 🏗️ Physical Design / Backend 정리
- 💾 Memory, Storage, Materials 기초
- ✅ UVM / Verification 학습 페이지
- 🔌 On-Chip Bus, Off-Chip Interface 정리
- 🤖 AI Accelerator 개념
- 📝 예상 기출문제와 실전 모의문항

## 🛠️ 로컬에서 보기

브라우저에서 `index.html`을 열면 바로 확인할 수 있습니다.

```bash
python -m http.server 8000
```

실행 후 `http://localhost:8000`으로 접속하세요.
