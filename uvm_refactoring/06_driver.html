Transfer Transaction to DUT Signals

м¶”мғҒм Ғмқё Transaction к°қмІҙлҘј мӢӨм ң н•ҳл“ңмӣЁм–ҙмқҳ Pin-level мӢ нҳёлЎң ліҖнҷҳн•ҳм—¬ DUTлҘј кө¬лҸҷн•ҳлҠ” UVMмқҳ н•өмӢ¬ м»ҙнҸ¬л„ҢнҠёмһ…лӢҲлӢӨ.

[1] Driverмқҳ м—ӯн•  & [3] Active Agent

Active AgentлҠ” мӢңл®¬л Ҳмқҙм…ҳм—җм„ң 'кө¬лҸҷ(Drive)'кіј 'лӘЁлӢҲн„°л§Ғ(Monitor)'мқ„ лҸҷмӢңм—җ мҲҳн–үн•ҳлҠ” м—җмқҙм „нҠёмһ…лӢҲлӢӨ. мқҙ м•Ҳм—җм„ң DriverлҠ” SequencerлЎңл¶Җн„° л°ӣмқҖ мғҒмң„ л ҲлІЁмқҳ Transaction(мҳҲ: AXI/APB Read/Write нҢЁнӮ·)мқ„ н•ҙм„қн•ҳм—¬, м§Җм •лҗң н”„лЎңнҶ мҪң мҠӨнҺҷкіј нҒҙлҹӯ(Clock) нғҖмқҙл°Қм—җ л§һкІҢ мӢӨм ң н•ҳл“ңмӣЁм–ҙмқҳ л¬јлҰ¬м Ғ н•Җ мӢ нҳё(0кіј 1)лЎң ліҖнҷҳн•ҙ DUTм—җ мқёк°Җ(Pin Wiggle)н•©лӢҲлӢӨ.

рҹ’Ў Key Insight: TransactionмқҖ "мЈјмҶҢ 0x100м—җ лҚ°мқҙн„° 0xFFлҘј мҚЁлқј"лқјлҠ” S/Wм Ғ м§ҖмӢңлқјл©ҙ, DriverлҠ” мқҙлҘј "Clock Rising Edgeм—җм„ң PADDR=0x100, PWRITE=1, PWDATA=0xFFлЎң 1мӮ¬мқҙнҒҙ мң м§Җн•ҳлқј"лҠ” H/Wм Ғ лҸҷмһ‘мңјлЎң лІҲм—ӯн•ҳлҠ” нҶөм—ӯмӮ¬ м—ӯн• мқ„ н•©лӢҲлӢӨ.

[2] uvm_driver мғҒмҶҚкө¬мЎ°

м§Ғм ‘ м„Өкі„н•ҳлҠ” мӮ¬мҡ©мһҗ л“ңлқјмқҙлІ„лҠ” л°ҳл“ңмӢң uvm_driverлҘј мғҒмҶҚл°ӣм•„м•ј н•©лӢҲлӢӨ. мқҙл•Ң, #(REQ, RSP) нҢҢлқјлҜён„°лҘј нҶөн•ҙ SequencerмҷҖ мЈјкі л°ӣмқ„ Transactionмқҳ нғҖмһ…мқ„ лӘ…мӢңн•©лӢҲлӢӨ. (мқјл°ҳм ҒмңјлЎң RSPлҠ” мғқлһөн•ҳм—¬ REQмҷҖ лҸҷмқјн•ҳкІҢ мӮ¬мҡ©н•©лӢҲлӢӨ.)

class my_driver extends uvm_driver #(my_transaction);

[4] Interface vs Virtual Interface

лҸҷм ҒмңјлЎң л©”лӘЁлҰ¬м—җ н• лӢ№лҗҳлҠ” Class(S/W)лҠ” н•ҳл“ңмӣЁм–ҙ л°°м„  лҚ©м–ҙлҰ¬мқё м •м Ғ Module(H/W) InterfaceлҘј м§Ғм ‘ к°Җм§Ҳ мҲҳ м—ҶмҠөлӢҲлӢӨ. л”°лқјм„ң virtual interfaceлқјлҠ” 'нҸ¬мқён„°(н•ёл“Ө)'лҘј нҶөн•ҙ л¬јлҰ¬м Ғ м„ л“Өм—җ м ‘к·јн•©лӢҲлӢӨ. мқҙ л‘ҳмқ„ мқҙм–ҙмЈјлҠ” л§Өк°ңмІҙк°Җ л°”лЎң uvm_config_dbмһ…лӢҲлӢӨ.

[5] м—°кІ° нғҖмқҙл°Қ & [6] build_phase & [7] run_phase

Driverмқҳ мғқлӘ…мЈјкё°лҠ” мІ м ҖнһҲ л¶„лҰ¬лҗ©лӢҲлӢӨ. мӨҖл№„ лӢЁкі„мқё build_phaseм—җм„ң н•ҳл“ңмӣЁм–ҙмҷҖ м—°кІ°н•  нҸ¬мқён„°лҘј лҜёлҰ¬ м„Өм •н•ҳкі , мӢӨм ң мӢңл®¬л Ҳмқҙм…ҳ мӢңк°„мқҙ нқҗлҘҙлҠ” run_phaseм—җм„ңлҠ” л¬ҙн•ң лЈЁн”„лҘј лҸҢл©° мӢ нҳёлҘј м§ҖмҶҚм ҒмңјлЎң кө¬лҸҷн•©лӢҲлӢӨ.

// [6] build_phase: Get Virtual Interface
function void build_phase(uvm_phase phase);
  super.build_phase(phase);
  // н•ҳмң„ м»ҙнҸ¬л„ҢнҠё(Driver)лҠ” к°’мқ„ л°ӣм•„мҳөлӢҲлӢӨ(get)
  if (!uvm_config_db#(virtual bus_if)::get(this, "", "vif", vif))
    `uvm_fatal("NO_VIF", "Virtual interface not found")
endfunction
// [7] run_phase: Active Loop
task run_phase(uvm_phase phase);
  // мӢңл®¬л Ҳмқҙм…ҳ лӮҙлӮҙ мў…лЈҢлҗҳм§Җ м•ҠлҠ” л¬ҙн•ң лЈЁн”„
  forever begin
    seq_item_port.get_next_item(req); // 1. Wait & Get
    drive_one_item(req); // 2. Drive to DUT
    seq_item_port.item_done(); // 3. Handshake Done
  end
endtask

Driver Transaction Loop л©”м»ӨлӢҲмҰҳ

SequencerмҷҖ Driver мӮ¬мқҙмқҳ нҶөмӢ мқҖ л§Өмҡ° м—„кІ©н•ң Handshake н”„лЎңнҶ мҪңмқ„ л”°лҰ…лӢҲлӢӨ. мҡ”мІӯ(Req)мқ„ л°ӣм•„ мІҳлҰ¬н•ҳкі , мҷ„лЈҢ(Done)лҘј м•ҢлҰ¬кё° м „к№Ңм§Җ лӢӨмқҢ TransactionмқҖ м „лӢ¬лҗҳм§Җ м•ҠмҠөлӢҲлӢӨ.

[8] seq_item_port

Driver лӮҙл¶Җм—җ лӮҙмһҘлҗң(built-in) TLM нҸ¬нҠёмһ…лӢҲлӢӨ. Sequencerмқҳ seq_item_exportмҷҖ м—°кІ°(Connect)лҗҳм–ҙ л‘җ м»ҙнҸ¬л„ҢнҠё к°„мқҳ кі мҶҚлҸ„лЎң м—ӯн• мқ„ н•©лӢҲлӢӨ.

[9] get_next_item()

Sequencer лӮҙл¶Җ FIFOм—җ мғҲлЎңмҡҙ Transactionмқҙ л“Өм–ҙмҳ¬ л•Ңк№Ңм§Җ лҢҖкё°(Blocking)н•©лӢҲлӢӨ. м•„мқҙн…ңмқҙ мӨҖл№„лҗҳл©ҙ мқҙлҘј к°Җм ёмҷҖ лЈЁн”„мқҳ лӢӨмқҢ лӢЁкі„лҘј м§„н–үн•©лӢҲлӢӨ.

[11] drive_one_item()

мӮ¬мҡ©мһҗк°Җ м§Ғм ‘ кө¬нҳ„н•ҳлҠ” User Taskмһ…лӢҲлӢӨ. `req` к°қмІҙмқҳ лҚ°мқҙн„°лҘј Virtual Interfaceмқҳ н•Җм—җ мқёк°Җн•ҳл©°, нҒҙлҹӯ м§Җм—°(`@(posedge clk)`) л“ұ мӢӨм ң мӢңл®¬л Ҳмқҙм…ҳ нғҖмһ„мқ„ мҶҢлӘЁн•©лӢҲлӢӨ.

[10] item_done()

н•ҳл“ңмӣЁм–ҙ кө¬лҸҷмқҙ мҷ„лІҪнһҲ лҒқлӮ¬мқҢмқ„ Sequencerм—җ м•ҢлҰҪлӢҲлӢӨ. мқҙ н•ЁмҲҳк°Җ нҳём¶ңлҗҳм–ҙм•јл§Ң Sequencerк°Җ лё”лЎңнӮ№мқ„ н’Җкі  лӢӨмқҢ Transactionмқ„ ліҙлӮј мҲҳ мһҲмҠөлӢҲлӢӨ.

[12] HW Rule мӨҖмҲҳ л°Ҹ Signal Control Task

мӢӨл¬ҙм—җм„ңлҠ” drive_one_item() лӮҙм—җ лӘЁл“  лЎңм§Ғмқ„ н•ҳл“ңмҪ”л”©н•ҳм§Җ м•ҠмҠөлӢҲлӢӨ. лҢҖмӢ , н”„лЎңнҶ мҪң мҠӨнҺҷ(APB, AXI л“ұ)мқҳ м—„кІ©н•ң HW к·ңм№ҷ(Setup/Hold мӢңк°„, Handshake, Burst л“ұ)мқ„ м§ҖнӮӨкё° мң„н•ҙ к°ңлі„ Task(мҳҲ: drive_write, drive_read)лЎң л¶„лҰ¬н•ҳм—¬ н•Җ л ҲлІЁмқҳ м •л°Җн•ң мӢ нҳё м ңм–ҙ(Signal Control)лҘј мҲҳн–үн•©лӢҲлӢӨ.

// Dispatcher: лӘ…л №м–ҙм—җ л”°лҘё Task л¶„кё°
task drive_one_item(my_tx req);
  case (req.cmd)
    WRITE: drive_write(req);
    READ : drive_read(req);
  endcase
endtask
// HW Rules: м •л°Җн•ң Signal м ңм–ҙ л°Ҹ Handshake
task drive_write(my_tx req);
  // 1. Clock Sync (Setup time ліҙмһҘ)
  @(posedge vif.clk);

  // 2. Drive Data & Valid High
  vif.valid <= 1'b1;
  vif.data <= req.data;

  // 3. Wait for Handshake (HW Rule)
  wait(vif.ready == 1'b1);

  // 4. Clear signals after drive
  @(posedge vif.clk);
  vif.valid <= 1'b0;
endtask
