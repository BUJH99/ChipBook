DT

EDA Infra

정의. 설계팀이 툴과 서버를 안정적으로 쓰도록 EDA 실행 환경과 자동화 인프라를 구축하는 역할입니다.

주요 업무
  • EDA 툴 셋업과 라이선스 운영
  • 서버, 배치 환경, 설계 farm 관리
  • 반복 업무 자동화와 flow 지원

Flow 위치. 특정 한 단계보다 모든 설계팀을 가로지르는 공통 인프라 축입니다.

EDA Tool License Server Farm Automation

CS

Chiplet / Package

정의. Chiplet, Si-interposer, HBM, 고속 인터페이스 SI/PI까지 다루는 첨단 패키징 및 시스템 연결 축입니다.

주요 업무
  • Multi-die / chiplet 구조 기획
  • Si-interposer, package 연동 검토
  • 고속 채널 SI/PI와 인터페이스 분석

Flow 위치. PD 이후 패키지·시스템 축으로 이어지며, HPC/AI 플랫폼 문맥과 자주 연결됩니다.

Chiplet Si-Interposer HBM SI / PI

SC

Foundation IP

정의. Standard Cell Library 개발과 특성화를 맡는 Foundation IP 세부 직무로, 합성과 STA의 출발점을 만듭니다.

주요 업무
  • 기본 logic cell 설계와 PPA 최적화
  • timing / power / area 특성화
  • library release와 공정 버전 관리

Flow 위치. RTL보다 앞에서 공통 라이브러리를 만들며 synthesis·STA·PD의 바닥을 형성합니다.

Standard Cell Characterization Liberty PPA

MC

Memory Compiler

정의. SRAM Compiler와 memory macro, 주변 회로, 옵션 특성화를 담당하는 Foundation IP 세부 직무입니다.

주요 업무
  • SRAM macro / compiler 구조 설계
  • redundancy, low-power 옵션 제공
  • 메모리 주변 회로와 특성화 지원

Flow 위치. SC와 함께 Foundation IP 축을 이루며, RTL 이전에 재사용 가능한 메모리 자산을 제공합니다.

SRAM Compiler Memory Macro Redundancy Low Power
