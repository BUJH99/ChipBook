AP (Application Platform)

시스템 소프트웨어
정의
역할

설계된 칩이 실제 제품에서 동작하도록 펌웨어, BSP, 드라이버, 보안 환경을 구현하는 영역입니다.

핵심 포인트

하드웨어 설계 결과를 실제로 부팅되고 제어되는 시스템으로 완성합니다.

주요 업무
  • Cortex-M/A 기반 Bare-Metal 펌웨어와 부트 코드 개발
  • Secure Boot, ARM SystemReady, Trusted Firmware 기반 보안 환경 구축
  • Linux Kernel BSP와 Device Driver 개발, Automotive 환경 최적화
  • PCIe, DDR, Ethernet, USB 3.0 등 고속 인터페이스 검증과 보드 디버깅
Flow Stage

실리콘 브링업 이후 하드웨어 플랫폼 위에서 시스템 소프트웨어를 구동하고, 실제 제품 동작 환경을 완성하는 단계입니다.

Bare-Metal Secure Boot Linux BSP / Driver PCIe / DDR / USB3.0
