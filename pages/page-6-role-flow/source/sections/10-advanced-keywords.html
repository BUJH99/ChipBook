심화 키워드 정리

Test / Firmware Keywords
DFT
MBIST

Memory Built-In Self Test. SRAM 같은 내장 메모리를 칩 내부 로직으로 직접 검사해 불량을 찾는 구조입니다.

DFT
BIRA / BISR

Built-In Redundancy Analysis / Repair. MBIST가 찾은 메모리 불량 위치를 분석하고 spare row/column으로 복구하는 메모리 수율 향상 기법입니다.

DFT
IJTAG

IEEE 1687 기반 내부 계측 접근 표준입니다. 칩 내부의 센서, 테스트 포인트, MBIST 제어 블록을 외부에서 구조적으로 접근하게 해줍니다.

DFT
LBIST

Logic Built-In Self Test. 메모리가 아니라 조합/순차 로직 쪽을 칩 내부 패턴 발생기와 시그니처 분석기로 검사하는 자체 테스트 방식입니다.

DFT
ATPG

Automatic Test Pattern Generation. stuck-at, transition 같은 결함 모델을 기준으로 양산 테스트용 패턴을 자동 생성하는 단계입니다.

AP
Linux BSP

Board Support Package. 특정 SoC와 보드에서 Linux가 부팅되도록 커널 설정, 드라이버, device tree, 초기화 코드를 묶은 시스템 소프트웨어 기반입니다.

AP
Bare-Metal Firmware

운영체제 없이 CPU가 전원 인가 직후 바로 실행하는 초기 펌웨어입니다. 부트 시퀀스, 클럭/메모리 초기화, 주변장치 bring-up, 보드 디버깅의 출발점으로 자주 연결됩니다.
