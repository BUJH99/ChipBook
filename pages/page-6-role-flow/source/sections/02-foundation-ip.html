역할

최첨단 반도체 공정을 바탕으로 Foundation IP를 설계·개발하는 영역입니다.

핵심 포인트

Standard Cell Library와 Embedded Memory처럼 모든 설계팀이 공통으로 활용하는 기반 자산을 만듭니다.

주요 업무
  • Standard Cell Library 설계와 PPA 최적화
  • Embedded Memory 및 SRAM 계열 매크로·컴파일러 개발
  • 공정 특성화와 라이브러리 검증, 설계팀 배포

Flow Stage. RTL 설계에 앞서 공통 설계 자산을 준비하며, 이후 합성·STA·PD 전반의 기준이 되는 기반을 제공합니다.
