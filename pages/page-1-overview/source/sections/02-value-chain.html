Memory Semiconductors

DRAM·SRAM 같은 휘발성 메모리와 Flash(NAND/NOR) 중심의 비휘발성 메모리가 함께 시장을 구성합니다. 반복되는 Array 집적도와 함께 메모리 계층 구조, 인터페이스, 전력 관리가 핵심 설계 포인트입니다.

RAM / NVM DRAM / NAND HBM Density
System Semiconductors (Logic/SoC)

연산과 제어를 담당하는 디지털 회로설계의 중심 축입니다. RTL 아키텍처, 기능 검증(DV), 합성, 저전력 통합 기술이 가장 활발하게 적용됩니다.

RTL Architecture DV / IP Subsystem AI Accelerator
Analog / Power / RF

디지털 시스템이 실제 물리 환경과 소통하도록 돕는 혼합신호 영역입니다. 고전압/전류 제어(PMIC)와 고속 통신(SerDes), 노이즈 제어가 중요합니다.

PMIC ADC/DAC SerDes / PLL
Sensors / Discretes

빛, 온도, 모션 등 물리적 세상의 정보를 감지하여 전기 신호로 변환합니다. 이미지 센서(CIS)의 화질 개선 및 내장형 ISP 논리 회로 설계가 포함됩니다.

CMOS Image Sensor MEMS ISP
Advanced Packaging / Chiplet

무어의 법칙 한계를 넘기 위해 여러 다이와 고대역폭 메모리(HBM)를 하나의 기판 위에 시스템처럼 묶는 차세대 핵심 기술입니다. Die-to-die 인터페이스, 열 분산, 신호 무결성 확보가 디지털 및 물리적 설계의 성패를 가릅니다.

Value 01 Design Owner
Fabless
역할 설계와 마케팅에 집중하며, 아키텍처 및 논리 회로(RTL) 최적화의 주체입니다.
산출물 Spec, RTL, Netlist(Synthesis), 제품 로드맵
대표 기업 Qualcomm, NVIDIA, AMD, MediaTek
Value 02 Bridge & Backend
Design House
역할 Fabless의 논리 회로를 Foundry의 특정 공정에 맞게 물리적 배치(P&R)하고 Signoff 검증을 대행하는 백엔드 전문가입니다.
산출물 Physical Layout, Final GDS, STA/DRC Report
대표 기업 Faraday, Alchip, 가온칩스, 에이디테크놀로지, 에이직랜드
Value 03 Tools & Reuse
IP / EDA
역할 설계 툴과 검증된 IP 블록을 제공해 칩 개발의 진입장벽을 낮추고 Time-to-Market을 단축합니다.
산출물 RTL IP, Hard Macro, Synthesis/P&R Toolchain
대표 기업 Arm, Synopsys, Cadence, Siemens EDA
Value 04 Manufacturing
Foundry
역할 팹리스/DSP가 넘긴 GDS 데이터를 기반으로 실제 웨이퍼를 가공 생산하며, 공정 PDK를 배포합니다.
산출물 Wafer, PDK, Design Rule, SPICE Model
대표 기업 TSMC, Samsung Foundry, UMC, GlobalFoundries
Value 05 Package & Test
OSAT (Packaging & Test)
역할 웨이퍼를 개별 칩으로 자르고 외부 기판과 연결한 뒤 출하 전 최종 테스트를 담당합니다.
산출물 완제품 패키지(SIP, MCM 등), Final Test 결과
대표 기업 ASE, Amkor, JCET, PTI, 하나마이크론
Value 06 IDM & Infra
IDM & Infrastructure
IDM 설계부터 공정 제조, 패키징까지 모두 내재화한 통합형 사업입니다. 대표 기업으로 Intel, Samsung, TI 등이 있습니다.
Infrastructure 노광/식각 장비 및 필수 특수가스, 웨이퍼 공급 생태계가 전체 산업의 토대를 이룹니다.
