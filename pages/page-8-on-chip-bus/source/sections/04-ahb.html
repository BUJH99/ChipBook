AHB Signal Descriptions

Signal Width Source Description
HCLK 1bit Clock source Bus clock. 모든 버스 전송은 HCLK의 상승 에지에 동기화됩니다.
HRESETn 1bit Reset controller Reset. 시스템과 버스를 리셋하는 Active LOW 신호입니다.
HADDR [31:0] Master Address bus. 32비트 시스템 주소 버스입니다.
HTRANS [1:0] Master Transfer type. 현재 전송의 유형(IDLE, BUSY, NONSEQ, SEQ)을 나타냅니다.
HWRITE 1bit Master Transfer direction. HIGH일 때 쓰기, LOW일 때 읽기 동작을 의미합니다.
HSIZE [2:0] Master 전송 데이터 크기(Byte, Halfword, Word 등)를 나타냅니다. (최대 1024비트 지원)
HBURST [2:0] Master Single, INCR, WRAP 계열 burst 유형을 나타냅니다.
HWDATA [31:0] Master 쓰기 데이터 버스입니다. 넓은 대역폭을 위해 더 확장될 수 있습니다.
HSELx 1bit Decoder 주소 디코더가 생성하며 특정 슬레이브가 선택되었음을 알립니다.
HRDATA [31:0] Slave 슬레이브가 구동하는 읽기 데이터 버스입니다.
HREADY 1bit Slave 버스 전송 완료를 나타내거나 추가 wait state를 요청합니다.
HRESP [1:0] Slave OKAY, ERROR, RETRY, SPLIT 같은 전송 결과 상태를 알려줍니다.

↺Same Master Retry
