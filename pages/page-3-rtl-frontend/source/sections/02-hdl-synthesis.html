HDL

Definition, Role, Output

Definition

Verilog, VHDL, SystemVerilog 기반 하드웨어 구조·동작 기술 언어, 소프트웨어 코드 형태를 빌린 회로 표현 수단

Role

대규모 회로의 코드 기반 설계·검증·합성 연결, RTL 수준 구조와 데이터 이동의 실제 설계 언어화

Output

RTL source 정리, simulation·synthesis 공통 입력 형성, 기능 검증과 gate-level netlist 생성의 출발 데이터

Logic Synthesis

Definition, Role, Output

Definition

RTL을 Standard Cell Library에 매핑해 Gate-level Netlist로 변환하는 과정

Role

공정 구현 가능한 논리 형태 확보, 성능·전력·면적(PPA) 목표 달성의 출발점 형성

Output

Netlist, timing report, area/power report, 이후 DFT·PD에서 사용하는 구현 기반 데이터
