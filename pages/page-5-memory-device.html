5-4. FinFET과 Planar

구조 차이, 등장 배경, 장단점 비교

미세화가 심해질수록 Planar 구조는 채널 통제력이 떨어져 누설 전류가 커집니다. 이를 개선하려고 게이트가 채널을 더 많이 감싸는 3차원 구조인 FinFET이 등장했습니다.

등장 배경

채널 길이가 짧아질수록 planar는 short-channel effect, DIBL, subthreshold leakage가 커져 제어 한계가 드러났습니다.

구조 핵심

Planar는 평면 채널 위를 gate가 누르고, FinFET은 솟아오른 fin의 윗면과 양옆을 gate가 감싸며 전계를 더 강하게 걸어줍니다.

장점

게이트 제어력 향상, leakage 감소, 낮은 전압에서도 더 좋은 스위칭 특성으로 성능·전력 효율을 함께 개선합니다.

단점

Fin 형성 공정, 패턴 정렬, layout 제약이 늘어 설계 규칙과 제조 난도가 더 높아집니다.
