1 Placement

Standard Cell을 빈 영역에 배치합니다. 실제로는 배선 가능성, 혼잡도, 타이밍, 전력, pin access를 함께 보며 위치를 반복적으로 다듬는 최적화 단계입니다.

세부 배치 및 최적화
Global Placement

전체 셀을 대략적으로 퍼뜨리며 density, wirelength, macro 주변 혼잡도를 먼저 맞추는 초기 배치 단계

density wirelength macro halo
Legalization

row/site 규칙에 맞게 셀 겹침 제거, 셀 정렬, keep-out 반영으로 실제 배치 가능한 형태로 정렬하는 단계

row/site overlap fix keep-out
Detailed Placement

국소 cell swap, reorder, micro-move로 타이밍과 배선성까지 세밀하게 다듬는 마무리 배치 단계

cell swap reorder micro-move
Congestion Relief

셀 spreading, whitespace 확보, macro channel 확보로 라우팅 막힘과 hotspot을 줄이는 혼잡도 최적화

spreading whitespace channel
Timing Optimization

critical path 주변의 resize, buffer insertion, VT swap, cell relocation으로 setup/hold 개선을 유도하는 타이밍 중심 최적화

buffer resize VT swap
Power / Routability

고밀도 영역 완화, switching hotspot 분산, 전원망 접근성 확보로 IR/EM과 후속 CTS·Routing 부담을 줄이는 배치 전략

PG rail pin access routing lane
2 CTS

플립플롭들에 클럭이 균형 있게 도착하도록 클럭 트리를 만듭니다. 목표는 Clock Skew와 Insertion Delay를 제어하는 것입니다.

세부 토폴로지
Clock Tree
Most Common Branching
버퍼 분기형 플롭 분포 대응 가장 보편적
가장 일반적인 버퍼 분기 구조. 플립플롭 분포에 맞춰 클럭을 나뉘어 보내며 skew 균형을 맞춥니다.
H-Tree

대칭성이 강한 구조. 칩 중심에서 사방으로 같은 길이로 퍼져서 이론적으로 skew 제어에 유리합니다.
Clock Spine

긴 중심축 측면 분기 긴 floorplan 유리
긴 중심 축(spine)을 먼저 깔고 가지를 뻗는 방식. 길쭉한 배치나 반복 구조 블록에서 자주 보입니다.
Clock Mesh
Dense Grid Network
격자형 강분배 skew 강함 전력·배선 비용 큼
격자형으로 클럭을 강하게 분배하는 구조. skew는 매우 강하지만 전력과 배선 자원이 많이 듭니다.
3 Routing

배치된 셀을 다층 금속 배선으로 연결합니다. 이 과정에서 배선 경로, via 개수, 층 선택, coupling, RC 증가가 결정되므로 최종 타이밍과 SI/PI 문제가 현실적으로 드러납니다.

세부 배선 단계
Global Routing

어느 영역과 어느 metal layer를 통해 갈지 큰 경로를 먼저 잡아 congestion과 routing resource 사용량을 예측하는 단계

corridor resource congestion
Track / Layer Assignment

배선 방향, 트랙, preferred direction, 상하 metal 선택과 via 위치를 정해 실제 연결 가능성을 구체화하는 단계

track layer via
Detailed Routing

각 핀을 실제 금속선으로 연결하면서 spacing, enclosure, via 규칙까지 만족시키는 정밀 배선 단계

metal path orthogonal rule clean
DRC Repair

minimum spacing, short, open, notch, antenna 같은 제조 규칙 위반을 reroute와 via 수정으로 줄여가는 보정 단계

spacing short/open reroute
RC / SI Aware Fix

긴 배선, coupling noise, crosstalk를 줄이기 위해 shielding, spacing, layer change, buffering을 적용하는 신호 무결성 최적화

crosstalk shielding spacing
Post-Route Optimization

추출된 RC 기준으로 setup/hold, IR/EM, slew, transition을 다시 점검하고 ECO까지 연결하는 라우팅 후반 최적화

RC extract STA ECO
물리설계에서의 위치 Floorplan 다음에 Placement/CTS/Routing이 오고, 이 결과를 바탕으로 Signoff 검증과 ECO 반복이 이어집니다.
