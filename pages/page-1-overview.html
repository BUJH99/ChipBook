Quick Notes
DDI
설명

Display Driver IC: smartphone·TV panel의 image data를 실제 pixel drive signal로 바꿔 화면을 구동하는 chip입니다.

FPLDs
설명

Field-Programmable Logic Devices: logic을 현장에서 다시 구성할 수 있어 prototyping이나 low-volume customization에 많이 쓰입니다.

T-CON
설명

Timing Controller: panel 내부에서 frame timing과 source/gate synchronization을 맞춰 display 흐름을 안정적으로 제어합니다.

PMIC
설명

Power Management IC: AP·memory·sensor에 필요한 voltage rail을 분배하고 charging, sequencing, protection을 담당합니다.

Memory Semiconductors

DRAM·SRAM 같은 휘발성 메모리와 Flash(NAND/NOR) 중심의 비휘발성 메모리가 함께 시장을 구성합니다. 반복되는 Array 집적도와 함께 메모리 계층 구조, 인터페이스, 전력 관리가 핵심 설계 포인트입니다.

RAM / NVM DRAM / NAND HBM Density
System Semiconductors (Logic/SoC)

연산과 제어를 담당하는 디지털 회로설계의 중심 축입니다. RTL 아키텍처, 기능 검증(DV), 합성, 저전력 통합 기술이 가장 활발하게 적용됩니다.

RTL Architecture DV / IP Subsystem AI Accelerator
Analog / Power / RF

디지털 시스템이 실제 물리 환경과 소통하도록 돕는 혼합신호 영역입니다. 고전압/전류 제어(PMIC)와 고속 통신(SerDes), 노이즈 제어가 중요합니다.

PMIC ADC/DAC SerDes / PLL
Sensors / Discretes

빛, 온도, 모션 등 물리적 세상의 정보를 감지하여 전기 신호로 변환합니다. 이미지 센서(CIS)의 화질 개선 및 내장형 ISP 논리 회로 설계가 포함됩니다.

CMOS Image Sensor MEMS ISP
Advanced Packaging / Chiplet

무어의 법칙 한계를 넘기 위해 여러 다이와 고대역폭 메모리(HBM)를 하나의 기판 위에 시스템처럼 묶는 차세대 핵심 기술입니다. Die-to-die 인터페이스, 열 분산, 신호 무결성 확보가 디지털 및 물리적 설계의 성패를 가릅니다.

GAA & CFET

3나노 이하 미세 공정에서 채널 4면을 게이트가 감싸는 구조로 전류 제어력을 높이는 기술입니다. 이후 CFET으로 진화하면서 디지털 설계 단계부터 새로운 전력/타이밍 모델 대응이 중요해집니다.

GAA (MBCFET) CFET Sub-3nm
PIM

메모리 안에 연산기를 직접 통합해 데이터 이동 전력을 줄이는 구조입니다. AI 가속기에서 폰노이만 병목을 완화하는 핵심 축으로 부상하고 있습니다.

Processing-in-Memory HBM-PIM AI Accelerator
CXL

CPU, GPU, NPU가 메모리 풀을 공유하도록 만드는 고속 인터페이스 표준입니다. 서버 시스템에서 메모리 확장성과 자원 유연성을 끌어올리는 핵심 기술입니다.

Memory Pooling Cache Coherency PCIe 5.0+
BSPDN

초미세 공정에서 신호 배선과 전력 배선을 분리해 혼잡도를 낮추는 2나노 이하 핵심 기술입니다. IR Drop 완화와 표준 셀 밀도 향상에 직접적인 효과가 있습니다.

Backside Power IR Drop Routing Congestion
On-Device AI / NPU

스마트폰과 자동차 같은 엣지 기기에서 직접 AI 연산을 수행하는 전용 칩입니다. 저전력 아키텍처와 데이터 재사용 최적화가 설계의 핵심 과제로 떠오르고 있습니다.

NPU Low Power Data Reuse
Automotive SDV SoC

수십 개의 ECU를 중앙 집중형 SoC로 통합하는 소프트웨어 정의 차량의 핵심 칩입니다. 성능뿐 아니라 ISO 26262, ASIL, fault tolerance까지 함께 검증해야 합니다.

Zonal Architecture ISO 26262 / ASIL Fault Tolerance
AiM / CiM

메모리 소자의 전도도를 가중치로 사용하고, 입력 전압을 인가해 행렬-벡터 곱셈을 아날로그 회로 수준에서 병렬 처리하는 구조입니다. 차세대 AI 가속기에서 에너지 효율을 크게 높일 수 있는 후보 기술입니다.

Analog CiM Crossbar Array MVM Acceleration
HBM

여러 DRAM 다이를 TSV로 수직 연결해 대역폭을 극대화한 메모리입니다. GPU/NPU와 같은 패키지에 탑재되어 AI 시대의 메모리 월을 해결하는 핵심 부품으로 자리 잡았습니다.

3D TSV Stacking High Bandwidth AI Accelerator
HBF

HBF는 NAND Flash를 HBM처럼 고대역폭 패키지로 재구성해 AI 추론 시스템의 메모리 용량 병목을 완화하려는 차세대 메모리입니다. HBM을 완전히 대체하기보다는 더 큰 용량과 비휘발성을 활용해 AI 가속기 옆의 보조 메모리 계층으로 주목받고 있습니다.

High Bandwidth Flash NAND-Based Memory Tier AI Inference
LPDDR

모바일과 AI PC, 온디바이스 AI 시스템에서 쓰이는 초저전력 DRAM 계열입니다. 높은 I/O 속도와 낮은 동작 전압, 대기 전력 최소화가 핵심이라 메모리 컨트롤러와 전력 관리 설계가 함께 중요해집니다.

LPDDR5X / LPDDR6 Mobile Memory On-Device AI
Emerging Memory

MRAM, ReRAM, PRAM 같은 신소자 기반 메모리는 DRAM의 속도와 NAND의 비휘발성 사이 간극을 메우려는 기술입니다. 임베디드 NVM, AI 가속기, 저전력 시스템에서 차세대 주력 후보로 검토됩니다.

MRAM ReRAM / PRAM Non-Volatile

Neuromorphic

SNN Event-Driven Ultra-Low Power
Silicon Photonics
