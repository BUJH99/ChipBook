9-1. Interface Stack

칩 내부 디지털 신호가 물리 보드 채널로 바뀌는 여정

Controller → PHY → Pad → Package → PCB → Endpoint

SoC 내부에서 완벽하게 제어되던 0과 1의 클린 디지털 데이터는 칩의 경계(Pad)를 넘어 서브스트레이트 패키지, PCB 보드 배선, 커넥터 등의 물리 채널(Physical Channel)을 거치면서 급격한 감쇠, 반사, 누설 등의 아날로그적 물리 현상에 노출됩니다. 고속 오프칩 인터페이스 설계는 이 물리적 불완전성을 극복하고 송수신단 사이의 전기적·타이밍 마진을 확보하는 극한의 혼성 신호(Mixed-Signal) 엔지니어링 영역입니다.

고속 오프칩 신호 전송 경로의 물리적 단면도 및 신호 왜곡 분석

Figure 9-1: 디지털 로직 도메인(SoC)에서 아날로그 채널 매체(패키지, PCB)를 통과하여 수신단(Target)으로 신호가 왜곡 및 회복되는 전 경로

📊 Off-Chip 인터페이스 스택 계층별 아키텍처 비교표

계층 이름 (Layer) 도메인 (Domain) 핵심 역할 (Role) 신호 무결성 위협 요인 (SI/PI Concerns) 설계 엔지니어링 체크포인트 (Checkpoints)
💻 Controller (논리 제어) 논리 (Logical) 프로토콜 제어, 패킷 빌딩, 에러 검출(CRC/Parity), 큐 매니지먼트 프로토콜 위반 (Protocol Violation), 대기 지연 (Latency) FIFO 오버플로우 방지, 상태 천이 예외 처리
🔌 PHY (물리 특성 계층) 혼성 신호 (Mixed-Signal) 병렬-직렬 변환(SerDes), 클럭 주입/복원(CDR), 등화기(EQ) 구동 ISI (심볼 간 간섭), 지터(Jitter), 셋업/홀드 타임 붕괴 DLL/PLL 락 시간 단축, 최적 Preset 등화 설정
⚡ PAD / IO (칩 경계 계층) 전기 (Electrical) 정전기 보호(ESD), 구동 능력 조절(Drive Strength), On-Die 종단 저항(ODT) 신호 반사(Reflection), 임피던스 불일치, 동시 스위칭 노이즈(SSN) 드라이버 매칭 저항 조절, 최적 전압 레벨 유지
📦 Package (패키지 물리) 물리 채널 (Physical) Bump 및 Ball 매핑, RDL 배선, 기생 성분 완화 기생 인덕턴스($L_{pkg}$), 커패시턴스($C_{pkg}$), 패키지 크로스토크 패키지 핀 배치 최적화, 기생 성분 모델링
📈 PCB Board (기판 전송) 물리 채널 (Physical) 마이크로스트립/스트립라인 배선, 비아홀(Via) 신호 전달 유전 손실, 도체 손실, 임피던스 불연속, 비아 반사파 특성 임피던스 매칭 ($50\Omega$ / $100\Omega$), 스큐(길이 차) 매칭

Protocol Layer

💡 패킷 단위의 에러 검출(CRC/Parity)과 재전송 프로토콜을 통제합니다. 고속 시리얼 통신에서는 물리 상태 머신인 LTSSM을 구동하고 링크 속도/대역을 동적으로 학습합니다.

Electrical Layer

⚡ SSTL, POD, LVDS 등 표준 규격에 부합하는 전압 스윙을 조절하고, 정전기 방지(ESD) 보호 소자를 거치며 On-Die Termination(ODT) 저항 매칭을 통해 반사파 노이즈를 억제합니다.

Timing Layer

⏱️ 패키지/보드 트레이스 길이 차이로 인해 채널마다 지연 시간이 달라지는 스큐(Skew)를 학습을 통해 보정(Training)하고, 랜덤/결정론적 지터 하에서도 충분한 Setup/Hold 타임을 획득합니다.
