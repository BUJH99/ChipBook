Chip Boundary

Off-Chip Interface는 칩 바깥 세상과 연결되는 관문입니다

메모리, 저장장치, 보드, 다른 칩과 통신하려면 외부 인터페이스가 필요합니다. 이 영역은 단순 protocol 이름 암기보다 controller + PHY + signal integrity + timing margin 관점으로 설명할 수 있어야 실무 감각이 살아납니다.

DDR / PCIe / USB / SerDes
Boundary Role
외부 장치 연결

DDR 메모리, SSD, 카메라, 다른 SoC, PC host 같은 외부 장치와 실제로 신호를 주고받습니다.

Key Layer
Controller + PHY

상위 protocol 처리 logic와 전기적 신호를 담당하는 PHY를 분리해서 보는 습관이 중요합니다.

Physical Risk
SI / PI / Timing

패키지, 보드, 채널 손실, jitter, skew, termination 조건까지 함께 묶어서 봐야 합니다.
